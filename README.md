# Laravel Project Base Template

## Dependencies

### Backend
- PHP 8.3, composer v2, mariadb 10 or mysql 8
- [Laravel 11](https://laravel.com/docs/)
- [Laravel Permissions](https://spatie.be/docs/laravel-permission/v6/introduction)

### Frontend
- [Vite v5](https://vitejs.dev/guide/)
- [HTMX](https://htmx.org/docs/)
- [Alpine JS](https://alpinejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/docs/installation)
- [DaisyUI](https://daisyui.com/docs/install/)
